== Install

[source,bash]
nimble install ski

== Usage

=== Simple examples

[source,nim]
----
import ski

echo "Sxyz".calculate(combinators)
----

Output is `xz(yz)` .

=== Other examples

See `examples` directory.
Run example code.

[source,bash]
nim c -r examples/calc_args.nim Sxyz

  xz(uz)

[source,bash]
nim c -r examples/read_file.nim

  Sxyz
    -> xz(yz)
  SKxz
    -> Kz(xz)
    -> z
  <true>xz
    -> Kxz
    -> x
  <false>xz
    -> SKxz
    -> Kz(xz)
    -> z
  SKIKSIKIKISKIKIS
    -> KK(IK)SIKIKISKIKIS
    -> KSIKIKISKIKIS
    -> SKIKISKIKIS
    -> KK(IK)ISKIKIS
    -> KISKIKIS
    -> IKIKIS
    -> KIKIS
    -> IIS
    -> IS
    -> S
